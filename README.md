# iLayer Solutions

Marketing site for iLayer Solutions — infrastructure, security, and automation
solutions for Telecom Operators, ISPs, and Enterprises. Built with Next.js and
Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

See `RESUME.md` for current project status, and `scripts/start-dev.sh` /
`scripts/stop-dev.sh` to run the dev server in the background.
